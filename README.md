
# Trackifly

App that shows stats from Spotify using their API 



## Installation

You can grab binaries from releases

Or build project:

```bash
  flutter doctor
  flutter pub get
  flutter build <--platform>
```




## Features

- Most played song, artist, genre
- History
- Full stats about most played songs, atrists and genres
- Material 3 design
- Cross platform
